# Incident Anomalies Alerts — index.html

## Overview
This file is the main HTML template for the Incident Anomalies Alerts web application. It provides the user interface for uploading videos, viewing results, and interacting with the detection system.

## Features
- Upload video files for anomaly detection
- View detected incidents and anomalies
- Display processed video frames and results
- User-friendly layout and navigation

## Requirements
- A running Flask backend (see README_app.md)
- Modern web browser (Chrome, Firefox, Edge, etc.)

## Usage
1. Start the Flask backend by running `python app.py`.
2. Open your browser and go to `http://localhost:5000`.
3. Use the interface to upload videos and view detection results.

## Notes
- The template is located in the `templates` folder.
- All video processing and detection logic is handled by the backend.
- The interface is designed for ease of use and clarity.
